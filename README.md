# postmark

Use Postmark API on the command line

## Installation

with [basher](https://github.com/basherpm/basher)

	$ basher install pforret/postmark

or with `git`

	$ git clone https://github.com/pforret/postmark.git
	$ cd postmark

## Usage

	> postmark.sh [action]
